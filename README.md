# Free Fire Max Overlay App

A lightweight overlay application for Free Fire Max with aimbot toggle and customizable menu.

## Features

- **Floating Overlay Menu** - Stays on top of the game window
- **Aimbot Toggle** - Easy on/off switch with visual feedback
- **Hotkey Support** - Press F6 to show/hide menu
- **Modern UI** - Clean, professional design with green theme
- **Lightweight** - Minimal resource usage
- **System Tray** - Run in background

## Hotkeys

- **F6** - Show/Hide Menu
- **F7** - Toggle Aimbot
- **F8** - Exit Application

## Requirements

- Python 3.8+
- PyQt5
- pynput

## Installation

```bash
pip install -r requirements.txt
python main.py
```

## Usage

1. Run the application
2. Press **F6** to toggle the overlay menu
3. Click the **Aimbot** button to enable/disable
4. Menu will appear in top-left corner of your screen
5. Settings are auto-saved
